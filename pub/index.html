Where can I find the documentation?

Documentation is available from the IBM webMethods Adapter Documentation Web site and from the Installer:

  • Visit the IBM webMethods area of the IBM Documentation Web site for the latest updates to the IBM webMethods Adapter documentation. The Web site also contains documentation for other products, documentation for previous versions of our products, and documentation in other languages.
  • As of version 9.0 of the IBM webMethods product suite, product documentation is also available for download from the IBM webMethods Installer. From the Installer, select "Documentation" to download and install the entire IBM webMethods product library, or navigate within the Documentation folder to select the product libraries to install. By default, the Installer installs the documentation files to a single directory called "_documentation," located under your IBM webMethods product installation directory.

1.3 Java Documentation

The IBM webMethods Adapter 10.1 for SAP uses the SAP Java IDoc Class Library to handle IDocs. However, creating, sending, and receiving IDocs is middleware dependent. Therefore, the IBM webMethods Adapter 10.1 for SAP provides a Java API that implements the interfaces defined by the SAP Java IDoc Class Library. The provided JavaDoc explains how to use that API.

  • IBM webMethods Adapter 10.1 for SAP IDoc Java API

3 Migration

The IBM webMethods Adapter 10.1 for SAP Installation Guide provides step by step instructions to migrate your existing IBM webMethods Adapter 4.6 for SAP based application packages to use IBM webMethods Adapter 10.1 for SAP. This is an easy to accomplish approach for an environment where you have defined a small set of IBM webMethods Adapter 4.6 for SAP constructs (for example, inbound maps) only.
